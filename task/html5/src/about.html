About This Project

A modern static website built with HTML5, CSS3, and JavaScript

Project Overview

This is a sample HTML5 static application designed to demonstrate modern web development practices using pure HTML, CSS (compiled from SCSS), and vanilla JavaScript.

Technologies Used

HTML5

Semantic markup with modern HTML5 elements

CSS3

Advanced styling with flexbox, grid, and animations

SCSS

Preprocessor for maintainable and organized styles

JavaScript (ES6+)

Modern JavaScript for interactivity

Build Process

The project uses a simple build process:

  1. SCSS files are compiled to CSS
  2. HTML and JavaScript files are copied to the dist folder
  3. All assets are organized for deployment
