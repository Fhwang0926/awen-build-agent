# Vue 3 Sample Application

This is a sample Vue 3 application for testing the build agent.

## Features

- Interactive counter component with Composition API
- Two-way data binding demo
- Modern gradient UI design with Vue 3 styling
- Responsive layout
- Built with Vite for fast builds
- Ready for production build

## Build Instructions

### Install Dependencies
```bash
npm install
```

### Development Server
```bash
npm run dev
```

### Production Build
```bash
npm run build
```

The build output will be in the `dist` directory.

### Preview Production Build
```bash
npm run preview
```

## Build Agent Integration

This project is designed to work with the LLM build agent system:
- **Framework**: Vue 3 (Composition API)
- **Build Tool**: Vite
- **Output Directory**: `dist/`
- **Build Command**: `npm install && npm run build`

## Vue 3 Features Demonstrated

- Composition API with `<script setup>`
- Reactive state with `ref()`
- Event handling
- Two-way binding with `v-model`
- Scoped styles
